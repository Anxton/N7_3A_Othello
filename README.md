# N7_3A_Othello

# Présentation (user doc ?)

Jeu Othello, se joue à 2 joueurs.

Règles :
* On met en place la partie comme suit : Image
* Alternativement, chaque joueur pose un pion de sa couleur :
  -> Chaque pion posé doit obligatoirement encadrer un ou plusieurs pions adverses avec un autre pion de sa couleur, déjà placé.
  -> Les pions encadrés changent alors de couleur
* La partie prend fin quand aucun des joueurs ne peut jouer

Le vainqueur est le joueur ayant le plus de ses pions sur le plateau.
